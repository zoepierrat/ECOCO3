# ECOCO3 Analysis Repository

This repository contains data analysis workflows comparing ECOCO3 satellite-derived carbon flux estimates against FLUXNET tower observations, with a focus on diurnal and seasonal WUE patterns under drought stress.

## Project Structure

```
ECOCO3/
├── README.md
├── plot_scripts.py                         # Visualization module (20 functions)
├── config.py                               # Shared configuration / paths
│
├── Reformat_FLUXNET_Data_Shuttle.ipynb     # Step 0 — reformat raw FLUXNET data shuttle
├── 00a_FLUXNET_datashuttle_preprocess.ipynb  # Step 1 — FLUXNET preprocessing & QC
├── 00b_ECOCO_preprocess_C1_V3.ipynb          # Step 2 — ECOCO3 preprocessing & QC
├── 01_Analysis_V5.ipynb                      # Step 3 — main analysis & figures
├── Access_ECOCO_GES_DISC.ipynb               # GES DISC data access helper
│
├── data/
│   ├── Support/                            # External reference datasets (not in git)
│   │   ├── koppen_geiger_0p1.nc            #   Köppen-Geiger climate classification
│   │   └── spei01.nc                       #   SPEI drought index
│   ├── ECOCO3_cleaned/                     # Processed ECOCO3 data (not in git)
│   └── FLUXNET_Data_Shuttle/               # Processed FLUXNET data (not in git)
│
├── figures/
│   ├── diurnal_cycles_drought/             # Per-group diurnal cycle plots
│   └── WUE_analysis/                       # Summary suppression & centroid figures
└── tables/                                 # CSV summary tables (generated by notebook)
```

## Pipeline

Run notebooks in order:

1. **`Reformat_FLUXNET_Data_Shuttle.ipynb`** — reads the raw FLUXNET data shuttle snapshot, attaches Köppen-Geiger climate class, and writes `complete_fluxnet_data_shuttle_metadata_table.csv`
2. **`00a_FLUXNET_datashuttle_preprocess.ipynb`** — QC, gap-filling, WUE computation for FLUXNET; writes half-hourly and daily CSVs + metadata
3. **`00b_ECOCO_preprocess_C1_V3.ipynb`** — QC and WUE computation for ECOCO3; attaches SPEI and climate class
4. **`01_Analysis_V5.ipynb`** — loads both datasets, filters to valid vegetation/climate groups, produces all analysis figures and summary tables

## plot_scripts.py — Function Reference

### Actively used in 01_Analysis_V5

| Function | Description |
|---|---|
| `plot_data_coverage_map()` | Spatial + temporal coverage map for ECOCO3 |
| `plot_data_coverage_map_sites()` | FLUXNET site coverage map + bar charts |
| `plot_diurnal_cycles_spei()` | Diurnal cycles stratified by drought / non-drought |
| `plot_diurnal_cycles_spei_comparison()` | FLUXNET vs ECOCO3 diurnal comparison by drought |
| `plot_merged_diurnal_cycles()` | Side-by-side diurnal curves with bootstrap CI |
| `plot_seasonal_cycles_comparison()` | 3-panel seasonal cycle — FLUXNET (solid) vs ECOCO3 (dashed) |
| `plot_violin_comparison_stacked()` | 4-panel WUE distribution comparison |
| `plot_drought_suppression_summary()` | Forest plots of midday Δ + 95% bootstrap CI by veg & climate |
| `plot_centroid_shift_summary()` | Bar charts of diurnal centroid timing shift under drought |

### Helpers / dependencies

`apply_lowess`, `compute_diurnal_centroid`, `get_stats`, `get_group_letters`, `hour_to_timestamp`, `plot_violin`, `_load_supp`, `_parse_h`, `_compute_shifts`, `_sig_star`

### Calling the two new summary figures

```python
import plot_scripts

# Uses tables/ CSVs generated by 01_Analysis_V5; saves to figures/WUE_analysis/
plot_scripts.plot_drought_suppression_summary()
plot_scripts.plot_centroid_shift_summary()

# Override paths if needed
plot_scripts.plot_drought_suppression_summary(
    sup_veg_path='tables/suppression_summary_midday.csv',
    sup_kg_path='tables/suppression_summary_midday_kg.csv',
    output_path='figures/WUE_analysis/drought_suppression_summary.png'
)
```
